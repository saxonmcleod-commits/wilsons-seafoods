## Features
*   **Admin Panel:** An intuitive admin interface that saves time by allowing easy updates to business hours and settings.
*   **Dynamic Content:** Real-time updates for "Today's Catch" and other critical business information.
*   **Premium Design:** A visually stunning, responsive interface with smooth animations and a focus on user experience.
*   **SEO Optimized:** Built with best practices to ensure high visibility on search engines.

## The Tech
Built with a modern, robust stack designed for performance and scalability:
*   **Supabase:** For a powerful, scalable backend and database.
*   **TypeScript:** Ensuring code reliability and maintainability.
*   **React & Vite:** For a blazing fast, interactive frontend.
*   **Tailwind CSS:** For rapid, beautiful styling.

## Run Locally

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Configure Environment:**
    Set the Supabase credentials in `.env.local`.
    ```bash
    cp .env.example .env.local
    ```
    You will need your Supabase Project URL and Anon Key.

3.  **Run the app:**
    ```bash
    npm run dev
    ```
